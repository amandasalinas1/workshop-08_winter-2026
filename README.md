# ENVS-193DS_workshop-08

## General information

What is this repo for?
This repo is for workshop 8, multiple linear regression, and demonstrating model selection using AIC.

Whose repo is it? Who is contributing to it?
This is a repo forked from the ENVS-193DS-2026 organization. I am the sole contributor.

## Data and file information

What files are there?
The data folder includes Valliere_etal_EcoApps_Data.xlsx, which is from: Valliere, Justin; Zhang, Jacqueline; Sharifi, M.; Rundel, Philip (2019). Data from: Can we condition native plants to increase drought tolerance and improve restoration success? [Dataset]. Dryad. https://doi.org/10.5061/dryad.v0861f7

What code is there, and what does it do?
The code folder includes workshop-code.qmd, which contains all code for fitting and comparing models.
